How to displaying a static page in Django

  • Project Creation: Initiate a new Django project.
  • App Creation: Create an app within the project.
  • Template Definition: Design a static HTML template.
  • Template Configuration: Set up template directories in settings.
  • View Definition: Implement a view function to render the template.
  • URL Routing: Establish URL patterns to connect views to URLs.
  • Request Handling: Django receives an HTTP request for the specified URL.
  • Response Generation: The view processes the request and renders the template.
  • Response Delivery: The rendered HTML is returned as an HTTP response to the client.
